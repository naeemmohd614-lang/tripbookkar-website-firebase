# **App Name**: Tripify

## Core Features:

- Hotel Data Storage: Store hotel details like name, brand, location, images, and facilities in Firestore.
- Package Management: Allow admins to create and manage travel packages with itineraries and pricing.
- Dynamic Pricing: Automatically adjust pricing based on demand and other factors using an API route.
- Image Processing: Automatically optimize and convert uploaded images to WebP format using a Cloud Function.
- Location Search: Allow users to search for hotels by city and state.
- SEO Page Generation: Automatically generate SEO-optimized pages for cities, states, and hotel types with the help of an LLM tool, to ensure only relevant data is included in the generated page
- Recommendations: Display AI-generated personalized hotel recommendations using user's past searches, leveraging the tool's awareness of popular destinations and current trends to produce insightful tips.

## Style Guidelines:

- Primary color: Deep blue (#1E3A8A) to convey trust and sophistication.
- Background color: Light gray (#F9FAFB), a very light tint of the primary color for a clean and modern feel.
- Accent color: Orange (#E67E22) to highlight key elements like buttons and links.
- Headline font: 'Playfair', serif, for elegant headlines. Body Font: 'PT Sans', sans-serif, for clear and readable body text.
- Use outline-style icons to maintain a modern and clean aesthetic.
- Implement a grid-based layout for responsive design and consistent spacing.
- Use subtle transition animations for a smooth user experience.
